# loss-cost-model-workshop
Hands-on actuarial modeling session materials

Welcome! This repository contains materials for the session:
**"Let's Build a Loss Cost Model Together!"**

## Files Included

'loss_cost_workshop.R' - main R script
'loss_cost_data.csv' - dataset

## Getting Started

1. Download the files:
   - Click the green **Code** button
   - Click **Download ZIP**
2. Install:
   - R
   - RStudio
3. Open the '.R' file in RStudio
4. Set working directory to the file location

## Requirements
- R (latest version)
- RStudio
- Packages: dplyr, ggplot2, caret, tweedie, statmod
